# Dommer

Super simple DOM operations. The API is listed below, and
works exactly like in [jQuery](https://github.com/jquery/jquery).

```javascript
const $ = import 'dommer';
const item = $('.item');
```

## API
* addClass
* item.data
* item.find
* item.hasClass
* item.on
* item.offset
* item.removeClass
* item.toggleClass
* item.width
